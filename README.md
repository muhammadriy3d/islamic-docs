# Getting started
## How to Install:
1. clone this repo
   ```bash
   git clone https://github.com/muhammadriy3d/islamic-docs.git
   ```
3. go into `islamic-docs/` directory
4. install npm packages:
   ```bash
   npm install
   ```
6. run:
   ```bash
   npm run dev
   ```
